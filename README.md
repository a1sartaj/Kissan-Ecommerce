# 🌾 Kisan Commerce Website

A modern, responsive **Kisan (Agriculture) E-commerce frontend** built using **React + TypeScript + Vite + Tailwind CSS**.

This project replicates the structure and experience of platforms like BigHaat, focusing on clean UI, performance, and scalability.

---

## 🚀 Features

* ✅ Responsive Navbar with Mega Menu
* ✅ Hero Section with animation (Framer Motion)
* ✅ Categories Section
* ✅ Trending Products
* ✅ Today’s Offers
* ✅ Shop by Pest & Disease
* ✅ Shop by Crop
* ✅ Best Selling Products
* ✅ Reusable Product Sections (Seeds, Sprayers, etc.)
* ✅ Blog / Articles Section
* ✅ SEO-friendly Footer
* ✅ Fully Responsive (Mobile-first design)

---

## 🛠️ Tech Stack

* **Frontend:** React + TypeScript
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Icons:** Lucide React + React Icons

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/a1sartaj/Kissan-Ecommerce.git
```

### 2️⃣ Navigate to frontend

```bash
cd Kissan-Ecommerce/frontend
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run development server

```bash
npm run dev
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview build:

```bash
npm run preview
```
